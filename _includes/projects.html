БАНКОВСКИЕ СИСТЕМЫ И СЕРВИСЫ

В рамках опыта ГК мы накопили значительную экспертизу в разработке банковских продуктов, с 2014 года.

Некоторые из наших проектов:

  • Разработка и сопровождение системы ДБО (интернет-банк) для корпоративных клиентов.
  • Разработка и сопровождение системы для работы с кредитными и страховыми договорами физлиц (кредитный конвейер).
  • Проектирование и разработка сервиса расчета пакетных продуктов банка.
  • Проектирование и разработка системы автоматизации искового производства юридического отдела банка.
  • Формирование кредитной истории на основе корпоративного хранилища данных для обмена данными с БКИ.
  • Разработка модулей секьюритизации кредитных договоров и обратного выкупа.
  • Разработка сервисов шопинг-карты и кредитного брокера.
  • Разработка систем сбора биометрических данных и внутренней системы банка для анализа показателей эффективности (KPI) сотрудников.
  • Интеграция корпоративного хранилища данных с платформой для рекрутинга.
  • Разработка утилит для сопровождения хранилища данных с набором инструментов для сборки и установки обновлений.

ФИНТЕХ / ИНВЕСТИЦИИ

В рамках инвестиционного направления имеется опыт реализации следующих проектов:

  • Витрина инвестиционных идей и продуктов. Позволяет инвесторам познакомиться и, при желании, тут же вложить свой капитал в уникальные торговые идеи, которые можно найти на рынке.
  • Фабрика инвестиционных идей. Агрегатор, инструмент создания и поставщик инвестиционных идей от аналитиков брокерских компаний.
  • Сервис оптимизации процесса продаж структурных продуктов для юридических лиц.
  • Первичное размещение паев. Создание и развитие сервиса по биржевому размещению паевых инвестиционных фондов.

МОБИЛЬНАЯ РАЗРАБОТКА

Мы обладаем значительным опытом в разработке мобильных приложений для платформ iOS и Android.

Несколько примеров реализованных проектов:

  • Приложение для проведения исследований с использованием георадиолокационного комплекса.
  • Приложение для мониторинга состояния заводского оборудования с поддержкой считывания RFID-меток и QR-кодов.
  • Приложение для взаимодействия диетологов и клиентов.
  • Приложение для взаимодействия тренеров и клиентов спортивных залов и фитнес центров.
  • Приложение для операторов животноводческих комплексов.
  • Приложение для поиска интересных мест для локального туризма.
